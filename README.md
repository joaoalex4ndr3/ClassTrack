# Fluxograma do Sistema ClassTrack

Este documento apresenta a explicação do fluxograma do **ClassTrack**, um Sistema Inteligente de Acompanhamento Escolar.

O fluxograma demonstra o funcionamento geral da plataforma, mostrando como cada tipo de usuário acessa o sistema, realiza login e utiliza as funcionalidades disponíveis de acordo com seu perfil.

## Visão Geral

O fluxo começa quando o usuário acessa a plataforma ClassTrack. Em seguida, ele escolhe seu perfil de acesso, podendo ser:

- Aluno;
- Professor;
- Direção.

Após escolher o perfil, o usuário realiza o login. Caso as credenciais estejam incorretas, o sistema exibe uma mensagem de erro e permite que o usuário tente novamente. Caso as credenciais estejam corretas, ele é direcionado para o painel correspondente ao seu tipo de acesso.

## Fluxo do Aluno

O aluno acessa a plataforma, seleciona o perfil **Aluno** e realiza o login.

Se as credenciais forem válidas, ele entra no painel do aluno, onde pode:

- Consultar frequência;
- Visualizar notas;
- Acompanhar seu desempenho acadêmico;
- Ver informações acadêmicas atualizadas.

Esse fluxo permite que o aluno acompanhe sua vida escolar de forma mais clara e organizada.

## Fluxo do Professor

O professor seleciona o perfil **Professor** e realiza o login.

Após a validação das credenciais, ele acessa o painel do professor, onde pode:

- Gerenciar turmas;
- Registrar frequência;
- Lançar notas;
- Registrar ocorrências;
- Acompanhar alertas acadêmicos.

Esse fluxo ajuda o professor a registrar e acompanhar informações importantes dos alunos de maneira mais prática.

## Fluxo da Direção

A direção escolhe o perfil **Direção** e realiza o login.

Com as credenciais corretas, o usuário acessa o painel da direção, onde pode:

- Acompanhar indicadores escolares;
- Visualizar relatórios;
- Monitorar o desempenho geral da escola.

Esse fluxo permite que a equipe gestora tenha uma visão mais ampla da situação acadêmica dos alunos e da instituição.

## Validação de Login

Em todos os perfis, o sistema verifica se as credenciais informadas são válidas.

Caso estejam incorretas, o sistema exibe uma mensagem de erro e retorna para a etapa de login.

Caso estejam corretas, o usuário segue para seu respectivo painel.

## Centralização dos Dados

Após as ações realizadas por alunos, professores e direção, os dados são centralizados e atualizados no sistema.

Essa centralização permite que as informações acadêmicas fiquem organizadas e disponíveis para os usuários conforme suas permissões.

## Continuação ou Encerramento

Depois de utilizar o sistema, o usuário pode escolher continuar navegando na plataforma.

Caso deseje continuar, ele permanece utilizando as funcionalidades do sistema.

Caso não deseje continuar, ele encerra a sessão por meio do logout.

## Encerramento

O fluxo termina quando o usuário realiza o logout e sai da plataforma.

Dessa forma, o fluxograma representa o funcionamento principal do ClassTrack, mostrando como o sistema organiza o acesso por perfis, valida as credenciais e direciona cada usuário para suas funcionalidades específicas.
