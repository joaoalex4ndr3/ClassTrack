# Banner ClassTrack

Este repositório contém o banner de apresentação do projeto **ClassTrack: Plataforma Web para Acompanhamento Acadêmico**.

O banner foi desenvolvido para a apresentação aberta da próxima semana, onde os projetos serão expostos para alunos, professores e demais participantes.

O objetivo da apresentação é mostrar a ideia central do ClassTrack, explicando o problema identificado, os objetivos do sistema, a metodologia utilizada, os resultados obtidos e as próximas melhorias possíveis.

Durante o evento, o banner servirá como material de apoio para que os professores possam avaliar as ideias, a organização e a proposta dos projetos desenvolvidos.

## Sobre o projeto

O **ClassTrack** é uma plataforma web criada para auxiliar o acompanhamento acadêmico de alunos, professores e direção escolar, centralizando informações como notas, frequência, ocorrências e indicadores de desempenho.

## Finalidade do banner

- Apresentar a ideia principal do projeto;
- Explicar o problema e a solução proposta;
- Apoiar a exposição durante a apresentação;
- Facilitar a avaliação dos professores;
- Mostrar os resultados e limitações do protótipo.
